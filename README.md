# Expert Curriculum Library

**고등학교 1학년 → 현업 전문가 수준**까지 끌어올리는 AI 생성 학습 자료 라이브러리.
문과 42개 + 이과 25개, 총 67개 분야 / 246개 단계 학습 콘텐츠.

---

## 누구를 위한 라이브러리인가

한국식 학교 교육의 틀 안에 있지만, 그 이상을 원하는 사람을 위해 만들어졌습니다.
교과서가 주지 않는 것 — **왜 그런지에 대한 논리적 추론, 현장 전문가의 언어, 스스로 생각하게 만드는 구조** — 을 담았습니다.

각 단계는 다음 흐름으로 구성됩니다:

1. **이론적 배경** — 7세 수준의 직관부터 대학원 수준의 개념까지 연속적으로 전개
2. **본문 설명** — 전문 용어 정의 + 논리적 추론 + 학술 출처 인용
3. **프로젝트** — 정답 없이 40분 분량의 예제 중심 실습, 평가 포함

단계당 학습 밀도는 **40분~1시간** 분량이며, 문단 형태의 줄글로 서술됩니다.

## 생성 방식

Claude Sonnet 4.6 모델이 단계별로 1회 호출되어 전체 콘텐츠를 생성합니다.
각 호출에서 두 개의 페르소나가 작동합니다:

- **Trainee** — 학생의 요청을 맥락과 함께 정교하게 재구성
- **Specialist** — 7세 → 고등학생 → 대학생 → 현장 전문가 순으로 단계적으로 설명하고, 전문 서적·논문을 인용하며 최소 3회 자체 검토
